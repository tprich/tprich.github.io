Natural Language Processing and Reddit

Two major applications of machine learning, webscrapping and natural language processing, were explored in this project. There were 1,500 posts collected from a couple of Reddit subreddits, r/Science and r/TodayILearned, using Pushshift's Reddit API. I then used natural language processing to determine which subreddit the post came from. Visit the Github to learn more.

Predicting Covid-19

My first group project, we decided to see which model was most effective at predicting the number of new daily Covid-19 cases between March 24th, 2021 and January 1st, 2022. We compared ARIMA and SIR models of predicting cases, and found that ARIMA tended to perform better. Visit the Github repo to learn more about both models and how the predictions were made. You can also check out our group's interactive Tableau dashboard to look at our actual predictions against the actual number of cases from the CDC.

Housing Prices - Linear Regression & Kaggle Competition

Ever try to guess how much a house will sell for? This project focused on creating a linear regression model to predict the sale price of a house in Ames, Iowa. Kaggle hosts this competition to see who could produce the best model. I only had two weeks to build my model. The best version used 135 features (the majority of which were dummy features) and won my cohort's version of this competition. Visit the Github for the project to learn more or click on the "Competition" button to visit the public version of the House Prices - Advanced Regression Techniques competition.
